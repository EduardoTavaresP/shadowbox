Thumbnail Gallery Example

This example demonstrates how to open a gallery of images in Shadowbox by clicking on a thumbnail of any image in the gallery. As in the simple thumbnail example each thumbnail is wrapped in a link element with a rel="shadowbox" attribute. Additionally, to ensure that Shadowbox displays all images in the same gallery, we include the name of the gallery that image belongs to between square brackets immediately after (e.g. rel="shadowbox[gallery-name]"). Then, when any image with the same gallery name is opened it will be opened alongside all other images in that same gallery.

Note: When using this method the gallery order will reflect the same order that the elements are found in the DOM.
